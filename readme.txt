Provide Wizit as a payment option for WooCommerce orders. Give your customer the option to buy now and pay later with 4 x interest free fortnightly instalments.

== Description ==

Give your customers the option to buy now and pay later with Wizit. The "Wizit Gateway for WooCommerce" plugin provides the option to choose Wizit as the payment method at the checkout. It also provides the functionality to display the Wizit logo and instalment calculations below product prices on category pages, individual product pages, and on the cart page. For each payment that is approved by Wizit, an order will be created inside the WooCommerce system like any other order.

== Installation ==

This section outlines the steps to install the wizit plugin.

> Please note: If you are upgrading to a newer version of the Wizit plugin, it is considered best practice to perform a backup of your website - including the WordPress database - before commencing the installation steps. 

1.  Login to your WordPress admin.
2.  Navigate to "Plugins > Add New".
3.  Type "Wizit" into the Keyword search box and press the Enter key.
4.  Find the "Wizit Gateway for WooCommerce" plugin. 
5.  Click the "Install Now" button.
6.  Click the "Activate" button.
7.  Navigate to "WooCommerce > Settings".
8.  Click the "Payments" tab.
9.  Click the "Setup" Wizit sub-tab.
10. Enter the Wizit API Key that was provided by Wizit for Production use. Setup merchant minimum and maximum payment amount.
11. Save changes.
